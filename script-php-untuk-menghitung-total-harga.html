** Mengapa Perlu Menghitung Total Harga?**

Menghitung total harga sering dibutuhkan dalam situasi bisnis, contohnya di toko online, di mana pelanggan membeli berbagai produk dan ingin tahu harga total yang harus dibayar. Dalam situasi tersebut, kita memerlukan script yang dapat membantu kita dalam menghitung total harga secara otomatis.

** Cara Menggunakan Script PHP untuk Menghitung Total Harga**

Untuk menggunakan script PHP di atas, kita perlu menyiapkan file PHP dengan coder/script PHP tersebut. Selanjutnya kita hanya perlu menjalankan file tersebut pada browser kita.

Berikut adalah langkah-langkah yang perlu kita lakukan:

  1. Buat file PHP dengan namamisal hitung_total_harga.php.
  2. Isikan kode script di atas ke dalam file tersebut.
  3. Simpan file tersebut.
  4. Jalankan file tersebut pada browser kita dengan mengetikkan http://localhost/hitung_total_harga.php (pastikan file tersebut berada di direktori http://localhost/).
  5. Tampilan total harga akan muncul di browser kita.
